## Harbor with Kubernetes: 


### 1. Determine Container Runtime:

_Run on the Kubernetes master node:_
```
kubectl get nodes -o wide
```


```
kubectl describe node worker1 | grep "Container Runtime"

Container Runtime Version:  containerd://2.2.5
```




### 2. Copy the Harbor CA to every Kubernetes nodes: 

_Copy `ca.crt` all worker nodes:_
```
scp /data/cert/ca.crt idea@worker1:/tmp

scp /data/cert/ca.crt idea@worker2:/tmp
```



### 3. Login each worker nodes:

_Path check:_
```
ll /usr/local/share/ca-certificates
```


```
cp /tmp/ca.crt  /usr/local/share/ca-certificates/harbor-ca.crt
```



_On each nodes:_
```
update-ca-certificates
```



### 4. Configure `containerd` registry trust:

_Create the directory:_
```
mkdir -p /etc/containerd/certs.d/reg.technbd.com
```


_Copy the CA:_
```
cp /tmp/ca.crt  /etc/containerd/certs.d/reg.technbd.com/ca.crt
```



_Create file `/etc/containerd/certs.d/reg.technbd.com/hosts.toml`:_
```
server = "https://reg.technbd.com"

[host."https://reg.technbd.com"]
  capabilities = ["pull", "resolve", "push"]
  ca = "/etc/containerd/certs.d/reg.technbd.com/ca.crt"
```




_Restart containerd:_
```
systemctl restart containerd
systemctl restart kubelet
```



### 5. Test with `ctr`:

```
ctr -n k8s.io images pull reg.technbd.com/library/node-app:1.4
```


```
ctr -n k8s.io images ls | grep reg.technbd.com/library/node-app:1.4
```
